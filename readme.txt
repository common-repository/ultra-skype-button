== Description ==

It's a awesome Skype button Plugin. You can make your own Skype call or chat button for your website. by this plugin you can customize button as you like. You can place button everywhere in your website by using Shortpre.

== Installation ==

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in Wordpress
3. Use shortpre in page or in widgets.
4. If you want Skype Button in your theme Place [ultra_skype btn_type="call" btn_id="1" color="white" user="theme.ultra" btn_size="32"] in your templates.

1. Shortpre 
<pre> [ultra_skype btn_type="call" btn_id="3" color="white" user="theme.ultra" btn_size="32"]</pre>


2. PHP pre 
<pre> <?php echo do_shortpre('[ultra_skype btn_type="call" btn_id="3" color="white" user="theme.ultra" btn_size="32"]'); ?></pre>
